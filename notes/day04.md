# 今日学习记录

## 今天完成的实验
1. 定时器中断切换闪灯
2. PWM 呼吸灯
3. SG90 舵机按键控制
4. ADC 读取芯片内部温度并串口输出

## 今天学到的内容
### 1. 定时器中断
我学会了启动基础定时器中断，也理解了可以通过修改 ARR 来改变中断周期。
另外，我知道了回调函数会在中断触发后执行，可以在里面翻转 LED。

### 2. PWM
我学会了使用 PWM 输出控制 LED 亮度变化。
占空比越大，LED 越亮。
如果循环修改 CCR，就能做出呼吸灯效果。

### 3. 舵机控制
我学会了用 PWM 控制 SG90 舵机角度。
按键可以作为输入，通过修改比较值控制舵机转动方向和角度。

### 4. ADC 和内部温度
我学会了 ADC 校准、启动转换、轮询读取数据，以及把原始值换算为电压和温度。
同时也练习了 printf 串口重定向。

## 遇到的问题
### 问题1
PWM 呼吸灯方向切换条件容易写错，可能导致只会越来越亮，不会变暗。

### 问题2
按键控制类实验如果不等待松手，就可能连续触发。

### 问题3
ADC 读取虽然能出数值，但换算公式还需要继续理解，后面还要再复习。

## 今日收获
今天的几个实验让我把定时器、PWM、按键、ADC、串口这些基础内容串起来了。
相比单独学一个外设，今天更像是在做完整功能。
