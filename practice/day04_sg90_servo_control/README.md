# SG90 舵机按键控制

## 功能
通过按键控制 SG90 舵机角度变化，按一个键增加角度，按另一个键减小角度。

## 使用内容
- TIM3 PWM 输出
- SG90 舵机控制
- GPIO 按键输入
- 占空比与角度映射

## 实现思路
1. 定义舵机最小值、最大值和步进值
2. 启动 TIM3_CH3 PWM 输出
3. 设置初始比较值
4. 检测 KEY1 按下后增加角度
5. 检测 KEY2 按下后减小角度
6. 每次修改后重新设置比较值

## 运行现象
按下 KEY1，舵机向一个方向转动。
按下 KEY2，舵机向相反方向转动。

## 遇到的问题
- 一开始不清楚舵机角度和 PWM 脉宽的关系
- 按键需要等待松开，不然会连续触发
- 步进值过大时，舵机会转得比较跳

## 总结
这个实验让我开始理解 PWM 不只是能调灯亮度，也能控制舵机角度。
