# Day03 UART Interrupt Echo

## 功能
使用串口中断接收数据，并将接收到的数据通过串口回显出来。

## 使用内容
- STM32 USART
- 串口中断
- NVIC 中断配置

## 实现思路
1. 初始化串口
2. 开启串口接收中断
3. 在中断回调函数中读取接收到的数据
4. 再通过串口发送出去，实现回显

## 运行现象
串口助手发送一个字符后，开发板会将该字符原样返回。

## 遇到的问题
- 一开始没有正确开启中断，导致收不到数据
- 没有重新开启接收，导致只能接收一次

## 总结
这个实验让我理解了串口中断接收的基本流程，也知道了回调函数是怎么配合工作的。
