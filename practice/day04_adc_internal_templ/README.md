# ADC 读取芯片内部温度

## 功能
使用 ADC 读取 STM32 芯片内部温度传感器数据，并通过串口打印温度值。

## 使用内容
- ADC1
- 芯片内部温度通道
- ADC 校准
- 串口 printf 重定向
- 温度换算公式

## 实现思路
1. 初始化 GPIO、ADC、定时器和串口
2. 先进行 ADC 校准
3. 启动 ADC 转换
4. 轮询等待转换完成
5. 读取 ADC 原始值
6. 将原始值换算为电压
7. 再根据公式换算为温度
8. 通过串口打印结果

## 运行现象
串口会持续输出当前芯片内部温度值。

## 遇到的问题
- 一开始不知道 ADC 使用前需要先校准
- printf 不能直接输出，需要先重定向到串口
- 温度换算公式需要注意参考电压和系数

## 总结
这个实验让我把 ADC、串口输出和数据换算串起来了，算是一个比较完整的小实验。
