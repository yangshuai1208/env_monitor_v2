# PWM 呼吸灯

## 功能
使用 TIM2 的 PWM 输出控制 LED 亮度变化，实现呼吸灯效果。

## 使用内容
- TIM2 PWM
- CCR 占空比调节
- LED 亮度渐变

## 实现思路
1. 初始化 TIM2 PWM
2. 启动多个 PWM 通道
3. 在循环中不断修改 CCR 值
4. 占空比逐渐增大再逐渐减小
5. 配合延时控制呼吸速度

## 运行现象
LED 会逐渐变亮，再逐渐变暗，形成呼吸灯效果。

## 遇到的问题
- 占空比边界判断容易写错
- 延时过小会导致变化太快，效果不明显
- 如果方向切换条件写错，LED 可能只会越来越亮

## 总结
这个实验让我理解了 PWM 的本质就是通过改变占空比来调节亮度。
