# Day04 Ultrasonic Distance Demo

## 功能
使用超声波模块测量目标距离，并通过串口输出当前距离值。

## 使用内容
- GPIO 输出触发信号
- TIM2 输入捕获
- USART 串口输出
- printf 重定向

## 实现思路
1. 初始化 GPIO、TIM2 和 USART1
2. 通过 TRIG 引脚发送一个短脉冲，触发超声波模块开始测量
3. 使用 TIM2 输入捕获记录回波信号的上升沿和下降沿时间
4. 计算高电平脉宽
5. 根据脉宽和声速换算出距离
6. 通过串口打印当前距离

## 运行现象
串口会周期性输出当前测得的距离值。

## 遇到的问题
- 一开始不清楚上升沿和下降沿分别对应什么时间点
- 输入捕获标志位如果不清除，可能会影响下一次测量
- 距离换算时要注意时间单位和声速单位是否一致

## 总结
这个实验让我理解了超声波测距的基本原理，也练习了输入捕获和串口打印的结合使用。
