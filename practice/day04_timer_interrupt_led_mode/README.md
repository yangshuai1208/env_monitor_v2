# 定时器中断切换闪灯

## 功能
按下按键后切换定时器中断周期，在定时器中断回调函数中翻转 RGB LED，实现不同频率的闪烁效果。

## 使用内容
- GPIO 按键输入
- TIM2 基础定时器
- 定时器中断
- 回调函数
- LED 电平翻转

## 实现思路
1. 先初始化 GPIO 和 TIM2
2. 启动 TIM2 中断
3. 在主循环中检测按键
4. 按键按下后切换模式
5. 根据模式修改 TIM2 的 ARR 值
6. 在定时器回调函数中翻转 LED

## 运行现象
上电后 LED 按固定频率闪烁。
按下按键后，LED 闪烁频率发生变化。

## 遇到的问题
- 一开始不清楚 ARR 修改后要不要重启定时器
- 按键长按会重复触发，所以需要等待按键松开

## 总结
这个实验让我理解了定时器中断、ARR 重装值和回调函数的基本用法。
