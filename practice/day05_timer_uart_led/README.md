# Day05 定时器 + LED + 串口整合实验

## 一、实验目标
在同一个工程中整合 GPIO、TIM1 和 USART2，实现一个简单的小系统：
- 定时器周期中断
- LED 周期闪烁
- 串口输出系统运行日志
- 按键切换闪烁速度

## 二、硬件资源
- 开发板：STM32F103C8T6
- LED：LED_R
- 按键：KEY1
- 串口：USART2
- TX：PA2
- RX：PA3
- 定时器：TIM1

## 三、功能说明
### 1. 定时器中断
使用 TIM1 产生周期中断。

### 2. LED 闪烁
每次定时器事件到来后，在主循环中翻转 LED_R 电平。

### 3. 串口日志输出
每次定时器事件到来后，通过 `printf` 打印当前运行状态，包括：
- 定时器次数
- 当前模式

### 4. 模式切换
按下 KEY1 后切换定时器 ARR，实现闪烁速度变化。

## 四、软件实现思路
本实验采用“中断置标志位，主循环处理”的方式：
1. TIM1 周期到达后进入中断
2. 在回调函数中设置 `timer_flag = 1`
3. 主循环检测到标志位后执行：
   - LED 翻转
   - 串口打印日志

这种方式可以减少中断中执行的内容，更规范，也更适合后续项目扩展。

## 五、关键代码
### 定时器中断回调
```c
void HAL_TIM_PeriodElapsedCallback(TIM_HandleTypeDef *htim)
{
    if (htim->Instance == TIM1)
    {
        timer_flag = 1;
        timer_count++;
    }
}
