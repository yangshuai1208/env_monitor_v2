# Day04 EC11 Encoder Demo

## 功能
使用 TIM3 编码器接口模式读取 EC11 旋转编码器的计数值，并通过串口打印旋转方向、步数和当前位置。
同时检测编码器按键按下事件，并在编码器静止超过 2 秒时输出提示信息。

## 使用内容
- TIM3 编码器模式
- GPIO 按键输入
- USART 串口输出
- printf 重定向

## 实现思路
1. 初始化 GPIO、TIM3 和 USART1
2. 启动 TIM3 编码器模式
3. 在主循环中读取编码器当前计数值
4. 将当前值和上一次的值比较，判断是否发生旋转
5. 通过定时器计数方向判断顺时针还是逆时针
6. 如果超过 2 秒没有移动，则输出静止提示
7. 检测编码器自带按键是否按下，并做简单消抖

## 运行现象
- 旋转编码器时，串口会输出旋转方向、移动步数和当前位置
- 编码器停止超过 2 秒后，串口会输出静止提示
- 按下编码器按键时，串口会输出按钮按下信息

## 遇到的问题
- 一开始不太清楚如何判断编码器旋转方向
- 按键没有消抖时，可能会重复触发
- 需要保存上一次计数值，才能判断本次变化量

## 总结
这个实验让我理解了编码器模式的基本使用方法，也进一步练习了串口调试输出。
