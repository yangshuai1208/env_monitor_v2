# Day06 ADC中断采集与串口打印实验

## 一、实验目标
本实验使用 STM32F103 的 ADC1 对两个传感器通道进行采集：

- 规则通道采集土壤湿度传感器电压
- 注入通道采集雨量传感器电压
- 将 ADC 原始值换算为电压值
- 通过 USART1 串口周期性打印结果

## 二、实验内容
本次实验完成了以下内容：

1. 初始化 GPIO、ADC1 和 USART1
2. 启动 ADC 校准
3. 使用 ADC 中断方式读取规则通道数据
4. 使用 ADC 注入通道中断方式读取第二路数据
5. 每 500ms 读取一次数据并换算为电压
6. 通过串口输出土壤湿度传感器和雨量传感器的电压值

## 三、工程说明
### 1. 串口配置
- 串口：USART1
- 波特率：115200
- TX：PA9
- RX：PA10

### 2. ADC采集思路
- `adc_val[0]`：保存规则通道 ADC 值
- `adc_val[1]`：保存注入通道 ADC 值
- `voltage[0]`：保存规则通道换算后的电压
- `voltage[1]`：保存注入通道换算后的电压

### 3. 主循环功能
主循环中完成以下流程：
- 启动一次规则通道 ADC 中断采样
- 启动一次注入通道 ADC 中断采样
- 延时 500ms
- 将 ADC 原始值换算成电压
- 使用 `printf` 通过串口打印结果

### 4. 中断回调函数
- `HAL_ADC_ConvCpltCallback()`  
  用于接收规则通道转换完成后的结果

- `HAL_ADCEx_InjectedConvCpltCallback()`  
  用于接收注入通道转换完成后的结果
