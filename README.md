# STM32 学习仓库

## 仓库说明
这是我在 STM32 第二阶段学习过程中的代码与笔记仓库，主要使用 STM32F103、HAL 库、CubeMX 和 Keil 进行练习。

当前阶段重点：
- GPIO 基础
- 串口发送与重定向
- 串口中断接收
- 定时器中断
- 多功能整合
- ADC 采集与串口打印

## 学习目标
通过 30 天 STM32 + FreeRTOS 学习计划，完成从基础外设到小型项目整合的训练，为后续秋招嵌入式岗位做准备。
